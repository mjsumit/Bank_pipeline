# End-to-End Banking Data Pipeline

This project simulates a relational banking data pipeline, combining programmatic data generation with relational database optimization.

## 📁 Project Structure
* **Bank_ipynb/**: Contains the Jupyter Notebook used to generate logical customer profiles and transaction ledger history.
* **Bank_Database_Setup/**: Contains the MySQL scripts, relational table schemas, and analytical views.
* **Bank_Sample_Dataset/**: Contains the generated raw customer and transaction CSV files.

## 🛠️ Tech Stack
* Python (Jupyter Notebook)
* MySQL Workbench
